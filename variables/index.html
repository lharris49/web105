Web 105

Web Fundamentals provides students with the tools to access, explore and contribute to the Internet. Students work with current web tools as well as create a basic website by hand coding.

This is Lenny's first semester at McHenry County College. He is pursuing an AA degree in Culinary Management, with a certification in Web Design. This is Lenny's first web design class, and he hopes to use his new found skills to assist nonprofit organizations in developing a more versatile and dynamic web presence.

Class Assignments

Introduction
This was our first introduction to html design. It is very simple, and just include some text and unformatted images
Text Formatting
In this second assignment, we were tasked with writing a review and incorporating more design elements, to include various text formatting options.
Images
This assignment was focused on using images. For example, how to use an image as a background and how to float images on the page.
Lists & Links
For this one, the focus was on creating various lists and links. On the lists page, we created several different types of lists. For the links page, we varied how the links opened up (in the current window vs. new tab for example).
Tables
This assignment showed us how to create and format various tables.
Project 1: Introduce Yourself
For this project, we brought together everything we had learned up to this point. It is a multi-page site that introduces ourselves to our classmates.
Blocks, Classes, & IDs
Here we were introduced to new elements we could include on our styles page to simplify coding on our index pages. We used the three elements (blocks, classes, and IDs) throughout the page.
Colors & Fonts
This is a demo page for the next project. Here we chose a color palette and a number of Google fonts to use in the Travel Site project. We also added some new element, like the "aside".
Project 2: Travel Site
Here is the final project. We had to first create wireframes for each page, then code the page to match.
Layout Tools
This assignment was all about creating a cascading style sheet we could use for future assignments. We set up our color scheme, font styles, and blocks, classes, and IDs (where applicable).
Variables
This is THIS page...and index for all out assignments. We starting incorporating variables onto our styles page, specifically with color schemes. This will make it easier to change the entire color scheme - or just parts - on an entire page quickly.
